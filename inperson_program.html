Who is this program for ?

This course is for those who are proficient/ basic in html,CSS and JavaScript, with familiarity in ES6 syntax, and want to excel in Node JS to become a back-end web development from SheCanCODE School:

WE ARE LOOKING FOR

  • Female Professionals starting out in their careers who want an entry point into the software development stack
  • Early-career developers, engineers, and other women software professionals, who want to expand their career opportunities
  • Non-technical professionals who want to switch careers to build websites Using Node Js
  • Independent and freelance developers and software professionals who want to expand their Backend Skills

What you will Learn

BUILD THE FOUNDATION

  • Understanding the key back-end concepts and how to approach design process
  • Writing clean and consider code
  • Understanding the lifecycle Node JS

BUILD CONFIDENCE

  • Build components and user interfaces using React
  • Hone skills by manipulating data and working with multiple components
  • Design, implement, and test front-end applications

PREPARE FOR JOB READINESS

  • Build expertise in back-end web design to create modern, secure web and mobile apps
  • Familizing your self with industry code management tools like Github and peer programming
  • Develop your professional portfolio to showcase your programming expertise

Program Fee

SheCanCODE is providing 100% scholarship for the finalists. ShecanCODE financial challenges should not be the biggest challenge for women to learn software development. However, the finalist will pay for registration and access to technology fee.

Non-Student Payment Plan

Non student registration and technology payment is 50000 rwf. The requirement is to pay 50% of payment and pay the remaining amount in installment.

Students package

Students will pay 20000 rwf and they can choose a payment plan of 5000 rwf/Month.

  • Familia using Github
  • Access to internet
  • Owns/access to computer
  • Willing to commit 3-4 hours/day learning coding
  • Able to pass basic HTML, CSS, and JS coding challenge
  • Female aged between 18-35 years old.
